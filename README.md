# 📝 Note Saver App

A full-stack note-taking web application built with Node.js, Express, and MySQL — featuring a stunning animated galaxy UI.

## ✨ Features

- 📝 Create, Read, Update and Delete notes
- 🔍 Real-time search through notes
- 🌌 Animated galaxy background with shooting stars
- 💜 Glassmorphism UI design
- 📱 Fully mobile responsive
- 🔒 Secure environment variables

## 🛠️ Tech Stack

- **Frontend** — HTML, CSS, JavaScript
- **Backend** — Node.js, Express.js
- **Database** — MySQL
- **Environment** — dotenv
- **Dev Tool** — Nodemon

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MySQL installed and running
